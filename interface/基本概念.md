interface是TS对对象类型的定义进行规定与限制的, interface自定义一种对象类型并规定它的格式和必须属性, 
如果一个对象被定义为该种类型, 那么它必须满足 interface定义的这种对象类型 的格式.

比如VNode类就有一种专属的VNode类型, 使用该类型的对象必须具备Vnode对象所具备的属性(当然也有可选属性, 需要在interface定义类型时使用特殊写法)
```typescript
interface Person {
    name: string
    age: number
}

const p1: Person = {
    name: 'lin',
    age: 18
}
```
